Cara Mendapatkan Kunci API Gemini Gratis

Kunci API diperlukan untuk fitur ekstraksi data. Anda bisa mendapatkannya secara gratis:

  1. Buka Google AI Studio.
  2. Masuk dengan akun Google Anda.
  3. Klik tombol "Create API key in new project".
  4. Salin (copy) Kunci API yang muncul.
  5. Tempel (paste) kunci tersebut di kolom di atas. Kunci akan disimpan di browser Anda.
